# Restaurant Tips Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Tips Dataset to understand customer tipping behavior.

## Objectives
- Analyze relationship between bill amount and tips
- Study tipping patterns across different days
- Understand the distribution of restaurant bills

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Google Colab

## Key Insights
- Higher bills tend to produce higher tips.
- Weekend days show higher tipping behavior.
- Most bills fall between $10 and $20.

## Dataset
Seaborn Tips Dataset

## Visualizations
- Scatter Plot
- Bar Plot
- Histogram
- Box Plot
- Heatmap
